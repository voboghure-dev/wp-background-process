== Description ==

This is a test plugin for check PHP Background Process built on WordPress/WooCommerce.

== Installation ==

= Minimum Requirements =

* PHP 7.2 or greater is recommended
* MySQL 5.6 or greater is recommended

= How to test =

Login to your WordPress admin panel and activate this plugin. Now add ?process=spall after your admin URL which will run a background process which by default write error log names getting from the function get_names().

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handles the file transfer, and you won’t need to leave your web browser. To do an automatic install of GWP Bundle Product, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”

In the search field type GWP Bundle Product,” then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the GWP Bundle Product plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
